# Virtualization Course - 2026 Edition

This repository contains the code examples and exercises for the Virtualization and Containerization course. Each exercise is located in its own directory (e1, e2, ..., eN) with a `README.md` file explaining the objectives and instructions.

## Prerequisites
- Docker installed on your machine. You can download it from [here](https://www.docker.com/get-started).
- Basic knowledge of command line operations.

## Exercises
- [Exercise 1: Create an image manually](e1/README.md)
- [Exercise 2: Add a file and set a command](e2/README.md)
- [Exercise 3: Create a Dockerfile and build an image](e3/README.md)
- [Exercise 4: Build, tag, and push an image](e4/README.md)
- [Exercise 5: Caching layers](e5/README.md)
- [Exercise 6: Multi-stage builds](e6/README.md)
- [Exercise 7: Bind volumes](e7/README.md)
- [Exercise 8: Volumes](e8/README.md)
- [Exercise 9: Network bridge](e9/README.md)
- [Exercise 10: Bridge and default bridge](e10/README.md)
